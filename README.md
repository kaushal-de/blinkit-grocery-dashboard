# 🛒 Blinkit Grocery Dashboard

## 📌 Project Overview
End-to-end data analysis project on Blinkit grocery delivery data
using Python for EDA and Power BI for interactive dashboard.

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI Desktop
- Jupyter Notebook

## 📊 Dashboard Pages
1. Overview — KPIs, Monthly Sales, Payment Method
2. Sales Analysis — Category, Day, Hour, Quarter
3. Product Analysis — Top Products, Brand, Margin
4. Customer Analysis — Segments, Areas, Order Value
5. Delivery Analysis — On Time %, Delay Reasons
6. Marketing Analysis — ROAS, Spend vs Revenue

## 🔑 Key Insights
- Total Revenue: ₹116.24M across 50K orders
- Peak Order Time: 12:00 AM
- Top Category: Instant & Frozen Food
- On-Time Delivery: 70%
- Repeat Customer Rate: 95%
- Best Marketing Channel (ROAS): SMS
- Top Area: Ghaziabad

## 📁 Dataset
Source: Kaggle — Blinkit Sales Dataset
